# Pathfinding
A* shortest pathfinding visualization.
        
Written by <i>SilverbossTD</i> (duynctsilverboss@gmail.com).

Language: <b>C++</b>.

Library: <b>SDL2</b>.

# Basic Controls

Create "Nodes":
   + Start: press 1 + left click.
   + End: press 2 + left click.
   + Wall: press 3 + left click.

Delete "Walls":
   + Press 3 + left click on the wall.
    
Start finding:
   + Press "Enter".
   
Clear the map:
   + Press "ESC".
